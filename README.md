**Svadilfari brings Gesture Control for Safari on iPhone/iPad.**

## Features

- Control Safari with Gestures
- 10+ Customizable Actions
- Select Patterns from Presets or Draw Your Own
- Exclusion List
- Set Gesture Sensitivity
