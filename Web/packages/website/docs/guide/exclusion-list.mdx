# Exclusion List

Use the exclusion list to disable gestures on specific domains or URLs.

To add an entry to the exclusion list, select Svadilfari from the menu on the address bar.

import menuImage from "./assets/menu.png";

<div style={{ textAlign: "center" }}>
  <img style={{ maxWidth: "300px" }} src={menuImage} />
</div>

If you want to disable gestures on all pages on this website, turn on the first toggle.

If you want to disable gestures on this specific page, turn on the second toggle.
