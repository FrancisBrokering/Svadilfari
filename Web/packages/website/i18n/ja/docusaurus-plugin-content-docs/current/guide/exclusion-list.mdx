# 除外リスト

除外リストを利用して、特定のドメインや Web サイトでジェスチャーを無効化できます。

除外リストに追加するには、Safari のアドレスバーからメニューを開き、Svadilfari を選択します。

import menuImage from "./assets/menu.png";

<div style={{ textAlign: "center" }}>
  <img style={{ maxWidth: "300px" }} src={menuImage} />
</div>

同じドメインの全てのページでジェスチャーを無効化する場合は、1 つ目のトグルをオンにします。

このページのみでジェスチャーを無効化する場合は、2 つ目のトグルをオンにします。

import popupImage from "./assets/popup.png";

<div style={{ textAlign: "center" }}>
  <img style={{ maxWidth: "300px" }} src={popupImage} />
</div>
